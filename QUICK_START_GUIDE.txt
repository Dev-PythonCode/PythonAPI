- Input: User career prompt (e.g., "How to become a software testing engineer?")
- Processing: NLP entity extraction â†’ 6-tier semantic matching â†’ Career path lookup

matched_career
  The best matching career path from the 70 available

match_score (0-400+)
  Higher = better match. 200+ = strong match

confidence (low/medium/high)
  Confidence level based on match score

description
  Role description and job outlook

core_skills
  Essential skills for the role (typically 10-20)

technologies
  Tech stack relevant to role (typically 5-15)

learning_path
  6-phase structured learning journey:
  Phase 1: Foundations (basics and prerequisites)
  Phase 2: Core Concepts (main topic fundamentals)
  Phase 3: Advanced Topics (specialized knowledge)
  Phase 4: Practical Application (hands-on projects)
  Phase 5: Real-world Projects (complex scenarios)
  Phase 6: Career Development (advancement to next level)

  Each phase includes:
  - Duration (weeks)
  - Topics (5-7 per phase)
  - Resources (courses, books, practice)
  - Projects (hands-on practice)

timeline_weeks
  Total time to complete learning path (at recommended pace)

effort_per_week_hours
  Recommended study hours per week

projects
  Real-world projects to build portfolio

career_progression
  Path from current role to senior/lead positions

job_market
  Current demand, growth rate, market insights

salary_range_usd
  Typical salary range (can vary by location/experience)

job_requirements_matching
  Integration point with job requirements endpoint
  Contains: skills suitable for job matching
